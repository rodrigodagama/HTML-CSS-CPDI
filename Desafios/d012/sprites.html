Sprites

Exercícios e desafios feitos para a aula de Sprites em LUA que é parte de um jogo de plataforma 2D que envolve uma nave espacial controlável, animações de personagens, e interações de toque na tela para mover o personagem.

Criação do Plano de Fundo:

É criado um plano de fundo da cena usando a função display.newImageRect(). O plano de fundo é uma imagem chamada "bg.png" que é dimensionada para ajustar a tela.

A posição do plano de fundo é definida para o centro horizontal (display.contentCenterX) e uma posição vertical específica (135).

Configuração da Barra de Status:

A barra de status do dispositivo é ocultada usando display.setStatusBar(display.HiddenStatusBar) para proporcionar uma experiência de tela cheia.

Carregamento e Reprodução de Música de Fundo:

Um arquivo de música chamado "odissey.mp3" é carregado usando audio.loadStream() e reproduzido em loop infinito com audio.play().

O volume da música é definido como 0.3.

Criação de Sprite de Personagem:

Um sprite de personagem é criado usando graphics.newImageSheet(). O arquivo de imagem "player4.png" contém uma série de quadros (frames) que representam as diferentes animações do personagem.

As animações são definidas em uma tabela chamada sprite1Animacao. Cada entrada na tabela define uma animação, incluindo o nome, quadro inicial, contagem de quadros, duração e se a animação deve ser repetida infinitamente (loopCount=0).

Criação e Configuração do Personagem:

Um objeto de personagem é criado usando display.newSprite() e vinculado ao sprite criado anteriormente (sprite1 e sprite1Animacao).

A posição inicial do personagem é definida.

O tamanho do personagem é ajustado usando xScale e yScale.

A sequência de animação inicial é definida como "parado", e a animação é iniciada usando player:play().

Criação de Sprite para Botões:

É criado um segundo sprite usando graphics.newImageSheet(). Este sprite contém uma série de quadros representando botões e ícones.

Diferentes partes do sprite são usadas para criar botões para mover o personagem para a esquerda e direita.

Criação e Configuração dos Botões de Controle:

Dois botões, "botaoEsquerda" e "botaoDireita", são criados usando display.newImageRect() e vinculados ao sprite de botões.

As posições iniciais dos botões são definidas.

Os eventos de toque são adicionados aos botões para controlar o movimento do personagem quando o jogador tocar ou arrastar os botões.

Funções de Controle de Movimento:

As funções moverEsquerda e moverDireita são definidas para manipular o movimento do personagem.

Quando o jogador toca no botão esquerdo, o personagem move-se para a esquerda e executa a animação "andando".

Quando o jogador toca no botão direito, o personagem move-se para a direita e executa a animação "andando".

Esse código cria uma cena de jogo básica com um plano de fundo, música de fundo, um personagem animado e controles de toque para mover o personagem para a esquerda e direita. É um bom ponto de partida para um jogo de plataforma 2D e pode ser expandido com mais recursos, níveis e desafios
